# Practice-03: Основы Python (Функции, Лямбда, ООП)

## Структура проекта

* `functions/`: Определение функций, аргументы по умолчанию, `*args`, `**kwargs` и возвращаемые значения.
* `lambda/`: Работа с анонимными функциями (включая использование с `map`, `filter`, `sorted`).
* `classes/`: Введение в объектно-ориентированное программирование, конструктор `__init__`, методы и переменные классов.
* `inheritance/`: Принципы наследования, переопределение методов, функция `super()` и множественное наследование.
